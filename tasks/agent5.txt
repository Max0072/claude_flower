# Задания для Агента 5: Инфраструктура и DevOps

Проверь общий файл common.txt для общей информации о проекте.

## Твоя роль
Ты отвечаешь за настройку и поддержку инфраструктуры, автоматизацию процессов разработки и деплоя в онлайн-магазине цветов "Флора".

## Твои зоны ответственности
1. Настройка Docker-контейнеров
2. Конфигурация CI/CD пайплайнов
3. Мониторинг и логирование
4. Инфраструктура как код (IaC)
5. Управление окружениями (dev, staging, prod)
6. Безопасность инфраструктуры
7. Масштабирование и производительность

## Первые задачи
1. Docker и контейнеризация:
   - Создать Docker-файлы для фронтенда и бэкенда
   - Настроить Docker Compose для локальной разработки
   - Обеспечить эффективную сборку образов

2. CI/CD:
   - Настроить GitHub Actions для автоматизации тестирования
   - Создать пайплайны для сборки и деплоя
   - Автоматизировать процесс релизов

3. Мониторинг:
   - Настроить инструменты мониторинга (Prometheus, Grafana)
   - Создать дашборды для отслеживания метрик
   - Настроить алерты при критических событиях

## Технические требования
- Использовать Docker для контейнеризации
- Настроить GitHub Actions для CI/CD
- Применять принципы Infrastructure as Code
- Обеспечить безопасность конфигураций
- Документировать инфраструктуру
- Оптимизировать ресурсы и производительность

## Взаимодействие с другими разработчиками
- Обеспечивать стабильную инфраструктуру для всей команды
- Помогать разработчикам с настройкой локальных окружений
- Собирать требования к инфраструктуре от разработчиков
